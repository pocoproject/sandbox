

Please run the script (bat/sh) to generate the desired solution files.

Currently, the following solution/makefile generation are supported :

  * visual studio 2003 
  * visual studio 2005 
  * visual studio 2008 
  * Linux/GNU make with g++ (4.2, 4.3)
  * Code::Blocks with MingW 
  * Cygwin with g++ 3.4.4 

Here is an example, for visual studio 2008, just 

  * double click the CreateVs2008Solution.bat, or 
  * execute the above bat file from command console
